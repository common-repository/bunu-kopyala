== Description ==

This is a plugin designed to create copies of pages or posts. You can quickly install it and freely copy as many posts and pages as you want.

Installation Instructions: Download and activate the plugin. You can start using it right away.

Usage: To copy your pages and posts, hover over the page you want to copy in the admin panel and click on the "Copy this" link to create your copy.
